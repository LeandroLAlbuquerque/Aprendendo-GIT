Terminal do Windowns:

echo >> '...' ------> aqui você ira criar um programa onde você quiser onde poderá colocar a formatação desejada
rm -----------------> remove o que você quiser
mv teste teste1 ----> Dessa forma você consegue renomear um nome que deseja
cd /c/.../.../.../--> Vai para o diretorio que deseja
touch ... ----------> cria um txt, muito utilizado para o gitignore



Terminal BASH(git):
git init -----------> inicia o git direto no diretorio que você quiser, criando assim uma "print" de tudo que você colocou ali
git add '...'-------> Adicina os programas que estão na pasta para seu git, isso é importante antes de você dar o commmit
git commit -m '...'-> Tira uma print do local, é como você deixasse salva uma versão, nela conterá tudo até aquele momento da pasta
git clone '...'-----> clona e colcoa aí um outro git que você tiver salvo para caso você queira usar, famoso crtl C + crtl + V
    ###Obs.: Nesse git clone você ainda consegue pegar coisas do git hub é só colocar a URL de onde você salvo do seu github
git status ---------> utilizado para ver se você não add algo, ou o que você fez naqueçla versão, o que mudou, o que apagou
git status -s ------> bem mais enxuto, e melhor para visualizar do que o primeiro, porém você tem que saber algumas terminologias:
    A : para modificados
    D : para os corretos, que não tiveram nenhuma alteração
    ? : para os novos arquivos que não foram adicionados no git
git log ------------> Mostra todas as versões do commit e suas sitaxes completas de HEXADECIMAL
git log --oneline --> Forma mais reduzida e melhorada para utilizar o log
